            바
            꿈

그 밖의 테스트를 다루는 태그들

아무 태그 없음 : 일반 글꼴
<b> : 단순히 글자를 굵게 표시하는 태그
<strong> : 글자를 강조하는 태그(웹 접근성)
<i> : 단순히 글자를 기울이는 태그
<em> : 글자를 강조하는 태그 (웹 접근성)
<blockquote> : 인용 문구를 나타내는 태그
- cite 속성 : 인용된 사이트 주소를 작성
사이트 주소가 직접 표시되진 않지만 검색 엔진에는 노출됨
<mark> : 형광펜 효과를 나타내는 태그
<u> : 글자에 밑줄을 긋는 태그
<small> : 글자를 작게 표시하는 태그
<sup> : 윗첨자를 나타내는 태그
<sub> : 아랫첨자를 나타내는 태그
<add> : 마우스 오버 시, 툴팁의 형태로 내용을 출력하는 태그
IOT란 각종 사물에 센서와 통신 기능을 내장해 인터넷에 결합하는 기술
<s> : 글자에 취소선을 넣는 태그

글자 관련 태그 응용

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은글자를 사용할 수도 있고, <기울이거나, 취소선을 넣는 등 다양하게 사용할 수 있다.
