div 태그와 span 태그의 차이점 1 : 줄바꿈

div 태그

div 태그 영역은 이미 존재하는 다른 태그 영역의 다음줄에 영역이 설정된다.
(block 형식 : 공간을 수직 분할함. width, height 속성 사용 가능)

첫 번째 영역
두 번째 영역
첫번 째 영역 다음줄에 영역이 설정됨
세 번째 영역
두 번째 영역 다음줄에 영역이 설정됨

span 태그

span 태그 영역은 줄바꿈이 일어나지 않고 옆으로 영역이 설정됨
단, 앞서 작성된 태그의 형식이 inline 형식일 때만 옆으로 설정됨 (inline 형식 : 공간을 수평 분할함. width, height 속성 사용 불가능)

첫번째 영역 두번째 영역 세번째 영역
