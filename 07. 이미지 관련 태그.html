이미지 관련 태그

            <img> : 웹 페이지에 사진이나 그림을 삽입할 때 사용하는 태그
            <img src="경로" alt="설명 문구" width="넓이" height="높이">

src 속성 : 이미지 파일의 경로를 작성하여 해당 이미지를 불러옴

alt 속성 : 사진의 경로가 잘못되어 이미지를 표현할 수 없을 때 대체 텍스트를 보여주는 용도의 속성

+시각장애인을 위한 화면 낭독기 프로그램을 위해 이미지에 대한 설명(텍스트)을 작성

width, height : 사진의 높이와 너비를 지정하는 속성

고정 크기 단위 : px
가변 크기 단위 : %, em, rm

고정 크기 단위 : 화면 사이즈가 변화해도 이미지 크기가 고정되어 변화하지 않음

가변 크기 단위 : 화면 사이즈 또는 이미지를 감싸고 있는 영역 사이즈에 따라 크기가 변화함
