폼 관련 태그

input 관련 태그 : html에서 사용자가 입력할 수 있는 양식을 제공하는 태그 form 태그 : 내부에 작성된 input 태그들을 통해 사용자로부터 데이터를 입력받아 서버로 넘기는 역할(전송) 

            form 태그 속성 
            *action 속성: form 태그 내부에 입력된 데이터를 전달받아 
                           처리할 페이지 또는 서버의 URL을 지정하는 속성
            *method 속성: 데이터 전송 방식을 지정하는 속성
                - get: URL(주소창)에 데이터를 담아 전달하는 방식(데이터를 볼 수 있음)
                       데이터 크기에 제한이 있음(256~4096byte)
                - POST: http request에 데이터를 넣어 전송하는 방식(데이터를 볼 수 없음)
                        데이터 크기 제한 없음
            *name 속성: form 태그의 고유 이름 지정. form 태그 구분을 위해 사용
            *target: action 속성에 작성된 페이지를 어떻게 열 것인지 지정

숫자와 관련된 input 태그
